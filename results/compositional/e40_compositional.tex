\subsection{End-to-End Compositional Verification}
\label{sec:compositional}

\noindent\textbf{Three-Tier Verification Architecture.}
We introduce a three-tier compositional verification
system that provides end-to-end formal guarantees for
KAN$\to$SCL compilation. The key insight is that
whole-program verification is decomposed into:
(1)~one-time compiler template proofs (Tier~1),
(2)~per-function Z3 verification of B-spline$\to$LUT
approximations (Tier~2), and
(3)~a machine-checkable composition certificate that
instantiates Theorem~1's structural induction (Tier~3).
The trusted computing base is a $\sim$200-line
certificate checker that verifies the composition
rules were correctly applied---it performs no Z3 solving,
only structural validation and arithmetic checks.

\textbf{Tier 1---Compiler Template Verification}
(One-Time).} We encode each IR operation's SCL code
template as a Z3 SMT formula with \textit{symbolic}
parameters and prove correctness for ALL possible inputs.
For MatMul, Add, and Argmax, Z3 returns
\texttt{UNSAT} on the negated equivalence query---the
SCL code is algebraically identical to the mathematical
specification. For BsplineLUT, we prove the linear
interpolation error bound $|f(x) - L(x)| \leq
M_2 h^2/8$ for any function with bounded second
derivative, mechanizing the de~Boor bound. For Softmax
and StandardAct, the analytic proof in
{\S}\ref{sec:method} applies (transcendental $\exp(x)$
prevents full Z3 mechanization, documented honestly).

\textbf{Tier 2---Per-Function B-Spline Verification}
(Instance-Level).} Each of the 
$512$ B-spline functions in the 
KAN $28x16x4$ model is independently verified 
by Z3: $\forall x\in[-3,3], 
|\text{LUT}(x) - \text{B-spline}(x)| \leq 
M_2 h^2/8$. All 
$512/512$ functions 
return \texttt{UNSAT}, confirming the per-function 
error bound of $arepsilon = 0.0036$.
Average verification time is 
$285$\,ms per function.

\textbf{Tier 3---Composition Certificate}
(Machine-Checkable).} The 
$9$-step composition
certificate instantiates Theorem~1's structural
induction, composing the per-function error bounds
through the IR graph via the Doubleton Arithmetic
propagation rules. The certificate is verified by
a $\sim$200-line trusted checker that validates:
(a)~all leaf certificates are present and verified,
(b)~each composition step's bound follows from its
inputs via the stated rule, and
(c)~the end-to-end bound is correctly computed.

\textbf{Key Insight.} The three-tier architecture
achieves whole-program formal verification without
requiring a monolithic SMT query over the entire
$3834$-line SCL program. Instead, it decomposes
the verification problem into:
(1)~one-time proofs about the \textit{compiler}
(independent of any specific model),
(2)~instance-level proofs about
\textit{individual components} (parallelizable), and
(3)~a lightweight \textit{composition check} that
combines (1) and (2) into an end-to-end guarantee.
This divide-and-conquer strategy is enabled by KAN's
structural properties: the decomposition into
independent univariate B-spline functions means
that component-level verification suffices for
whole-program correctness---a property that MLPs,
with their entangled multivariate activations,
cannot exploit.
