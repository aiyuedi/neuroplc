# NeuroPLC Figure Contract — Top-Journal Workflow

Target venue: IEEE Transactions on Industrial Informatics.

## Global export rule
- Final LaTeX includes only `paper/figures/final/*`.
- Every chart has traceable source data in `paper/figures/source_data/*`.
- MATLAB-native figures are generated by MATLAB and copied to `final/`.
- Origin-suited statistical charts use CSV as source and are rendered from an Origin-style MATLAB fallback until Origin COM is stable.
- Architecture diagrams remain diagram-native (TikZ/SVG), not MATLAB statistical plots.

## Quality gates
- No fake data: every number must appear in source CSV or generated formula section.
- No dual-axis for final data plots except where explicitly marked as diagnostic; prefer small multiples.
- Color must not be sole encoding; use line style/marker/hatching where applicable.
- All final figures exported to PDF vector + PNG preview; EPS where MATLAB print supports it.
